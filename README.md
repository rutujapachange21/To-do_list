If you have VS Code with Live Server extension:

1. Open the project folder in VS Code

4. The application will open at http://localhost:5500 (or similar)

## ğŸ“– How to Use

### Adding a Task

1. Type your task in the input field at the top
2. Press *Enter* or click the *"Add"* button
3. Your task will appear in the list below

### Completing a Task

1. Click on the task or the checkbox next to it
2. The task will be marked with a strikethrough
3. Click again to unmark it

### Deleting a Task

1. Hover over the task you want to delete
2. Click the *Delete* button (X or trash icon)
3. The task will be removed from the list

### Clearing All Tasks

- Click the *"Clear All"* button to remove all tasks at once (if available)

### Data Persistence

- All your tasks are automatically saved to your browser's local storage
- Tasks will remain even after closing and reopening the browser
- Clear browser data to reset the to-do list
